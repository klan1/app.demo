# k1.app
App skeleton using k1.lib library.
## Installation

```sh
git clone https://github.com/klan1/k1.app-skeleton.git
cd k1.app-skeleton
bower install
composer install
chmod +x ./controllers/install-app-scripts.sh
chmod +x ./controllers/update-app-scripts.sh
cd controllers
./install-app-scripts.sh
```
